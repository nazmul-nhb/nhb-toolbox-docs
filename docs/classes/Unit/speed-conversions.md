## kmphToMph()

### Signature

```typescript
static kmphToMph(kmph: number): number
```

### Return Type

`number` - Miles per hour equivalent

### Example

```javascript
Unit.kmphToMph(100); // 62.1371
```

## mphToKmph()

### Signature

```typescript
static mphToKmph(mph: number): number
```

### Return Type

`number` - Kilometers per hour equivalent

### Example

```javascript
Unit.mphToKmph(62.1371); // 100
```
