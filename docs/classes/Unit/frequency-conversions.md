## hzToKHz()

### Signature

```typescript
static hzToKHz(hz: number): number
```

### Return Type

`number` - Kilohertz equivalent

### Example

```javascript
Unit.hzToKHz(1000); // 1
```

## kHzToHz()

### Signature

```typescript
static kHzToHz(khz: number): number
```
