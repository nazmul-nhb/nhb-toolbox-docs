## joulesToCalories()

### Signature

```typescript
static joulesToCalories(j: number): number
```

### Return Type

`number` - Calories equivalent

### Example

```javascript
Unit.joulesToCalories(4.184); // 1
```

## caloriesToJoules()

### Signature

```typescript
static caloriesToJoules(cal: number): number
```

### Return Type

`number` - Joules equivalent

```javascript
Unit.caloriesToJoules(1); // 4.184
```
