## kgToLbs()

### Signature

```typescript
static kgToLbs(kg: number): number
```

### Return Type

`number` - Pounds equivalent

### Example

```javascript
Unit.kgToLbs(1); // 2.20462
```

## lbsToKg()

### Signature

```typescript
static lbsToKg(lbs: number): number
```

### Return Type

`number` - Kilograms equivalent

### Example

```javascript
Unit.lbsToKg(2.20462); // 1
```

## gramsToOunces()

### Signature

static gramsToOunces(g: number): number
