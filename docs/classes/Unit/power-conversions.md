## wattsToKw()

### Signature

```typescript
static wattsToKw(w: number): number
```

### Return Type

`number` - Kilowatts equivalent

### Example

```javascript
Unit.wattsToKw(1000); // 1
```

## kwToWatts()

### Signature

```typescript
static kwToWatts(kw: number): number
```

### Return Type

`number` - Watts equivalent

### Example

```javascript
Unit.kwToWatts(1); // 1000
```
